ES-IBD Explorer

ES-IBD Explorer controls all aspects of an ES-IBD experiment, including ion beam guiding and steering, beam energy analysis, deposition control, and data analysis.
Using the build-in plugin system, it can be extended to support additional hardware as well as custom controls for data aquisition, analysis, and visualization.
See the manual for more details.
